# PizzaBoy

A Unity game project written in C#.

## Setup
1. Install Unity Hub and Unity Editor (version 2021.3 or later recommended).
2. Open Unity Hub, go to Projects > Add, and select this folder.
3. Open the project in Unity Editor.
4. Use VS Code as your code editor: In Unity, Edit > Preferences > External Tools > External Script Editor > Browse to VS Code executable.
5. Write your C# scripts in the Assets/Scripts folder.

## Getting Started
- Attach the PizzaBoyController script to a GameObject in your scene.
- Build and run your game from Unity Editor.
